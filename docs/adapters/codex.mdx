title: Codex
description: "Add Pear Pro MCP to OpenAI Codex CLI."
---

Codex CLI configures MCP servers in `~/.codex/config.toml` under `[mcp_servers.*]`.

## 1. Build the server

```bash
cd ~/pear-pro-mcp && npm install && npm run build
```

## 2. Add the server

Edit `~/.codex/config.toml`:

```toml
[mcp_servers.pear-pro]
command = "node"
args = ["/Users/you/pear-pro-mcp/dist/index.js"]

[mcp_servers.pear-pro.env]
PEAR_API_BASE_URL = "https://hl-v2.pearprotocol.io"
PEAR_CLIENT_ID = "APITRADER"
PEAR_ADDRESS = "0x..."
PEAR_API_KEY = "..."
PEAR_READ_ONLY = "true"
```

## 3. Verify

Start Codex and ask it to use a tool:

> Call `pear_health` and then `pear_active_markets`; tell me today's top gainers.

<Note>
TOML is whitespace- and quote-sensitive. Keep every value (including `"true"`) quoted as a string, and use an absolute path for `args`.
</Note>
