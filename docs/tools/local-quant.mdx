## `pear_pair_analytics`

Compute correlation, beta (hedge ratio), spread z-score, rolling z-score and annualized vol from two aligned price series (oldest → newest). Mirrors the stats Pear's "Agent Pair" reasons over, fully offline — there is **no API call**.

<ParamField path="pricesA" type="number[]" required>Long-leg prices, oldest → newest (≥3 values).</ParamField>
<ParamField path="pricesB" type="number[]" required>Short-leg prices, same length & alignment (≥3 values).</ParamField>
<ParamField path="window" type="integer">Z-score lookback (default 30).</ParamField>
<ParamField path="periodsPerYear" type="number">Annualization factor: 365 daily, 8760 hourly (default 365).</ParamField>

`pricesA` and `pricesB` must be the same length and time-aligned, or the tool returns an error.

### Output

```json
{
  "n": 30,
  "correlationOfReturns": 0.78,
  "beta": 0.92,
  "spreadMean": 0.0541,
  "spreadStd": 0.0009,
  "latestSpread": 0.0565,
  "zScore": 2.31,
  "rollingWindow": 30,
  "rollingZScoreTail": [/* last 10 rolling z-scores */],
  "annualizedVolA": 0.62,
  "annualizedVolB": 0.55,
  "signal": "extended_long_spread"
}
```

| Field | Meaning |
| --- | --- |
| `correlationOfReturns` | Correlation of the legs' log returns |
| `beta` | OLS hedge ratio of A on B (`cov/var`) |
| `spreadMean` / `spreadStd` | Mean & std of the A/B ratio series |
| `latestSpread` | Most recent A/B ratio |
| `zScore` | Z-score of latest spread vs window; `|z|≥2` = mean-reversion signal |
| `rollingZScoreTail` | Last 10 rolling z-scores |
| `annualizedVolA/B` | Per-leg annualized volatility |
| `signal` | `extended_long_spread` (z≥2), `extended_short_spread` (z≤-2), or `neutral` |
