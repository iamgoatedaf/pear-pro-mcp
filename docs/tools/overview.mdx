## Result envelope

Every tool returns a single MCP `text` content block:

<CodeGroup>
```json Success
{ "content": [{ "type": "text", "text": "<JSON or string result>" }] }
```

```json Error
{ "content": [{ "type": "text", "text": "<message>" }], "isError": true }
```
</CodeGroup>

API errors are formatted as `Pear API error <status> on <endpoint>: <body>`. Guard rejections and validation failures are returned the same way with `isError: true`.

<Note>
"Guarded" tools require `PEAR_READ_ONLY=false` **and** `confirm: true` on the call. See [Safety](/get-started/safety).
</Note>
